# Tables

## HTML Tables

You can create them with: `<table></table>` tags.


```html
<table>
  <tr>
    <th>First Header</th>
    <th>Second Header</th>
    <th>Third Header</th>
  </tr>
  <tr>
    <td>This is a data cell</td>
    <td>This is also a data cell!</td>
  </tr>
    <tr>
    <td>This is a data cell</td>
    <td>This is also a data cell!</td>
  </tr>
    <tr>
    <td>This is a data cell</td>
    <td>This is also a data cell!</td>
  </tr>
</table>
```

```css
table, tr, th {
  border: 2px solid black;
}
```

The following is the notation for the creation of tables:
- `tr` -> table row
- `td` -> table data
- `th` -> table header
